🚚 FleetManagementApp
A robust Java-based solution for managing commercial vehicle lifecycles, logistics, and driver assignments.

📖 Overview
The FleetManagementApp is a comprehensive tool designed to streamline the complexities of fleet management. It provides a centralized system to track vehicle maintenance, manage driver schedules, and monitor operational efficiency. This project demonstrates core Object-Oriented Programming (OOP) principles, specifically focusing on how different entities (Drivers, Vehicles, and Schedules) interact within a software ecosystem.

✨ Key Features
Vehicle Inventory Tracking: Categorize and manage a diverse fleet including trucks, vans, and passenger vehicles.

Driver Assignment Logic: Efficiently link available drivers to specific vehicles based on license types and shift availability.

Maintenance Monitoring: Track mileage and service history to ensure fleet safety and longevity.

Data Consistency: Implements robust validation to ensure vehicle IDs and driver credentials follow standard formats.

🛠️ Tech Stack
Language: Java (JDK 17)

IDE: Apache NetBeans 16

Core Logic: Object-Oriented Design (Inheritance, Encapsulation)

Library Management: Integration of external JAR files (e.g., Vehicle.jar) for extended functionality.
